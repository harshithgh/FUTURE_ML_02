# FUTURE ML Task 02 - Support Ticket Classification

## 📌 Problem Statement

This project aims to automatically classify customer support tickets into categories and assign priority levels.

## ⚙️ Approach

* Cleaned and preprocessed text data
* Converted text into numerical format using TF-IDF
* Used Logistic Regression for classification

## 📊 Features

* Ticket category classification (Technical, Billing, General)
* Priority prediction (High, Medium, Low)
* Text preprocessing and tokenization

## 📈 Results

* Achieved good classification accuracy on test data
* Model evaluated using classification report

## 🧠 Skills Gained

* Natural Language Processing (NLP)
* Text vectorization
* Machine learning classification
